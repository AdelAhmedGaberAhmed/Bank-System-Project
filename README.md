# 🏦 Bank System (C++ Console Application)

## 📌 Overview
This is a console-based Banking System developed using C++. The project simulates real-world banking operations such as account management, transactions, currency exchange, and ATM withdrawals.

The system was designed and implemented using Object-Oriented Programming (OOP) principles to ensure scalability, maintainability, and clear separation of concerns.

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling / In-Memory Data Management (depending on implementation)
- Console-based UI

---

## 🏗️ System Design
The system is structured using OOP concepts such as:

- **Encapsulation** for securing user and account data  
- **Inheritance** for user roles (Admin / Customer)  
- **Polymorphism** for role-based behaviors  
- **Abstraction** to simplify system interactions  

---

## 👤 User Roles

### 🧑‍💼 Admin
- Manage users and accounts
- Approve or reject operations
- Control system permissions
- View system-wide data

### 👤 Customer
- Create and manage personal account
- Perform deposits and withdrawals
- View account balance and transactions
- Transfer money between accounts

---

## 💳 Key Features

- User registration and login system
- Secure authentication mechanism
- Deposit and withdrawal operations
- ATM-like withdrawal simulation
- Money transfer between different accounts
- Multi-currency support with conversion system
- Role-based access control (Admin / User)
- Console-based interactive menus

---

## 💱 Currency Exchange System
The system supports transactions between different currencies, allowing users to:
- Convert amounts between currencies
- Perform cross-currency transfers

---

## 🏧 ATM Simulation
- Withdraw money as if using a real ATM
- Balance validation before transactions
